Encrypted message:
How to read the message:
  1. Save encrypted message as msg.txt
  2. Backup cert: Edit -> Preferences -> Advanced -> Certificates -> View Certificates -> Your Certificates -> Backup... -> save as backup.p12
  3. Run commands: 
    	openssl pkcs12 -in backup.p12 -nocerts -out privateKey.pem
	base64 -d msg.txt | openssl rsautl -raw -inkey privateKey.pem -decrypt
